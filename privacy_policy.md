Privacy Policy — Shopping China USD Tax Free

Last updated: 2026-06-16

Summary
This browser extension runs only on pages on the shoppingchina.com.py domain and helps display estimated and exact USD (TAX FREE) prices by reading page content and using site search endpoints. It does not send data to external third-party servers owned by the developer.

What the extension reads
- Page content on pages matching https://www.shoppingchina.com.py/* (product titles, visible prices and snippets of page text) in order to compute USD equivalents and identify products.

What the extension stores locally
- Small cached values in Chrome local storage (chrome.storage.local):
  - An estimated exchange rate (rounded integer) with timestamp.
  - Per-product divisor cache entries used to compute exact USD values.
- These values remain on the user's machine until the extension is uninstalled or the user clears browser extension storage.

What the extension transmits
- Requests are made only to the same origin site endpoints used by the site itself (e.g., /quick_search?search=...), using same-origin credentials (cookies) when present. These requests are performed from the users browser directly to shoppingchina.com.py. The extension does not forward page content, prices, or any cached values to any external servers controlled by the developer.

Third-party services
- No analytics, crash-reporting, or third-party tracking services are used by the extension.

Security and code
- The extension does not execute remote code, does not eval user content, and avoids inline scripts to comply with extension platform security requirements.

User controls
- To stop data access, disable or remove the extension via chrome://extensions.
- To remove cached values without uninstalling: open DevTools → Application → Storage → Extension Storage (or clear browser data for the extension), or uninstall/reinstall the extension.
